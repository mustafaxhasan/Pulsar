# Pulsar

> A modular avionics and telemetry platform for mission management, sensor integration, real-time monitoring, and embedded systems development.

## Overview

Pulsar is designed as a simulation-first avionics platform that provides a foundation for telemetry processing, mission state management, data logging, hardware interfacing, and autonomous system experimentation.

The project follows a modular architecture, allowing components such as sensors, cameras, actuators, and communication systems to be developed and validated independently before full system integration.

## Current Features

- Mission State Machine
- Live Telemetry
- Sensor Simulation
- Data Logging
- Camera Interface
- Servo Interface
- Web Dashboard
- Technical Report System
